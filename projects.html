Jim Saves the World

Zombies survival wave based shooter game. Created in Unreal Engine 5.4.4 using blueprints. Version 1.01 of the game showcases the wave system, sound and visual effects and UI mangement.

Project role: Game Designer, developer, animator, UX/UI, VFX and sound design.

The Adventures of Felix

The Adventures of Felix is a retro style 2D platformer built using Unity. The project boasts achievement in level design, sound design, animation, platforming mechanics (moving platforms, lever-based platforms), a level select system and a boss fight.

Project role: Lead desinger, level designer, animator and developer.

Jetpack Madness

Jetpack Madness is a 3D platformer game built using Unreal Engine 5.4.4. This game is not for the faint of heart as the level can be punishing. The project includes the use of unreal systems to animate using blend spaces and state machines. Visual Effects for the jetpack thrusters and fuel pickups. Sound design and makes use of a fun platforming mechanic, pressure plates and moving platforms. The game was built using blueprints.

Project role: Developer, UI/UI, SFX, Level design, Animation.
